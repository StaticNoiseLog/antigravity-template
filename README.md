Antigravity Template
====================

This is a template for a new project using Antigravity.

It comes with three (very) opinionated playbooks implemented as workspace rules that describe three phases of a project:

1. requirements-engineering-playbook.md
2. solution-architecture-playbook.md
3. software-development-playbook.md

Usage
=====

Three workflows are provided that cor

- `/requirements`
- `/architecture`
- `/development`
